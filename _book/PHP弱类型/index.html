PHP弱类型

PHP 弱类型和魔术hashes

PHP提供了两种方法来比较两个数值：

  • 宽松的比较使用的是== 或者 != ：两个数值拥有相同的数值
  • 严格比较使用的是=== 或者 !==：两个数值有相同的数据类型和相同的数值

Type Juggling

True statements

bool类型

var_dump('0010e2'   == '1e3');           # true
var_dump('0xABCdef' == ' 0xABCdef');     # true PHP 5.0 / false PHP 7.0
var_dump('0xABCdef' == '     0xABCdef'); # true PHP 5.0 / false PHP 7.0
var_dump('0x01'     == 1)                # true PHP 5.0 / false PHP 7.0
var_dump('0x1234Ab' == '1193131');

'123'  == 123
'123a' == 123
'abc'  == 0

'' == 0 == false == NULL
'' == 0       # true
0  == false   # true
false == NULL # true
NULL == ''    # true

NULL statements

NULL类型

var_dump(sha1([])); # NULL
var_dump(md5([]));  # NULL
